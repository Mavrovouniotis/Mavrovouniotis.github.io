Ant Colony Optimization (ACO) Library for Dynamic Optimization Problems

 

This is an open source library that provides the implementation of all existing ant colony optimization (ACO) algorithms proposed for the dynamic travelling salesman problem (DTSP). The source code is provided for research purposes to provide an easy and efficient platform for everyone to compare and evaluate their algorithms.

The ACODTSP library is implemented in the C++ language and provides the implementation of three DTSP variations:

  • DTSP with weight changes
  • DTSP with node changes
  • DTSP with encoding changes

All the ACO algorithms proposed for DTSP in general are implemented for all three afformentioned DTSP variations. The algorithms include:

  • Shaking Technique [2]
  • Population-based ACO (P-ACO) [4]
  • Ant Colony System (ACS) with Maximum Trails [1]
  • Random Immigrant ACO (RIACO) [5]
  • Elitism-based Immigrant ACO (EIACO) [5]
  • Hybrid Immigrant ACO (HIACO-I) [5]
  • Interactive Hybrid Immigrant ACO (HIACO-II) [12]
  • Memory-based Immigrant ACO (MIACO) [7]
  • Environmental Information-based Immigrant ACO (EIIACO) [8]
  • MAX-MIN Ant System (MMAS) [14]
  • MMAS with Restart [5]
  • MMAS with Adaptive Evaporation [9]
  • MMAS with Self-Adaptive Evaporation [10]
  • Restart-Strategy [3]
  • eta-Strategy [3]
  • tau-Strategy [3]
  • Memetic P-ACO (M-PACO) [6]
  • Multi-colony MMAS [11]
  • Multi-caste ACS [13]

The ACODTSP library can be downloaded here: ACODTSP. Please let me know for any bugs/improvements/additions at: m.mavrovouniotis at hotmail.com
